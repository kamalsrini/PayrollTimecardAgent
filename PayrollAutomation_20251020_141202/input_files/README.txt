# Input Files Examples

## Excel Files
- Place Excel timesheet files (.xlsx, .xls) in this folder
- Files should contain 'time' in the filename
- Examples: "John Doe Time Sheet 10.13.25.xlsx"

## Image Files  
- Place timesheet images (.png, .jpg, .jpeg) in this folder
- Files should contain 'time' in the filename
- Examples: "Timesheet.png", "Employee Time Sheet.jpg"

## Supported File Types
- Excel: .xlsx, .xls
- Images: .png, .jpg, .jpeg, .bmp, .tiff

The automation will automatically detect and process all supported files.
